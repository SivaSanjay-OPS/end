LoanSphere

This is a frontend-only loan management system. No backend. All data stays in localStorage.

  • Admin — Review documents, accept/reject, oversee loans.
  • Lender — Create and publish loan offers.
  • Borrower — Apply, upload PDFs/Images, view payments.
  • Analyst — Visual charts & loan statistics.
  • HR — Manage user roles.
